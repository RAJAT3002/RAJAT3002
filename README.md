</div>

<details>
<summary><b>🔬 RISC-V SoC + CNN Accelerator</b></summary>

**Stack:** Verilog · RISC-V ISA · FPGA · Python golden reference

</details>

<details>
<summary><b>📡 Underground OFDM Channel Model (WUSN)</b></summary>

> Physics-based soil dielectric model generating 100K realistic underground channel samples. Trained deep neural net achieving **94% R²** in BER prediction. Benchmarked A*, Dijkstra, Bellman-Ford — A* wins with **83.7% path reliability**.

**Stack:** Python · Deep Learning · OFDM · Peplinski model

</details>

<details>

> Encoder + decoder in Verilog for up to **3-bit error correction** with UART. Completed RTL → GDSII: synthesis (Genus), P&R (Innovus), LEC, Gate-Level Sim, DRC/LVS clean.

**Stack:** Verilog · Cadence Genus · Innovus · Python · TCL

</details>

<details>
<summary><b>🏗️ Two-Level Cache Memory System</b></summary>

> L1 direct-mapped + L2 4-way set-associative FIFO cache in Verilog. L1–L2 miss-handling logic. PyQt6 GUI to visualise hit/miss ratios and cache performance metrics.

**Stack:** Verilog · Vivado · PyQt6

</details>

<details>
<summary><b>🔍 AMBA AHB Protocol Verification</b></summary>

> Full modular SystemVerilog UVM environment for AMBA-3 AHB-Lite slave. Layered testbench: transaction → generator → driver → monitor → scoreboard. Constrained random stimulus, SVA assertions, burst & arbitration corner cases.

**Stack:** SystemVerilog · UVM · SVA

</details>

<details>
<summary><b>🏎️ Formula EV Safety Systems — Team Chimera</b></summary>

**Stack:** KiCAD · Altium · Analog Devices BMS · Embedded C

</details>

<details>
<summary><b>🤖 eYRC 2024-25 — EcoMender Bot (IIT Bombay)</b></summary>
